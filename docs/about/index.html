关于本站

一个以官方资料和可复核边界为优先的中文信息站。

最后核验：2026-07-10 · 维护者：JumorHack

Claude 使用指南由 JumorHack 独立维护，目标是帮助读者分清 Claude 应用订阅、Console/API 账单和 Claude Code 的不同用途。本站不是 Anthropic 官方站，也不代表或暗示获得 Anthropic 的授权、合作或背书。

我们做什么

  • 把容易混淆的官方规则和产品边界用中文整理为可阅读的决策页；
  • 为每个关键页面给出对应的官方来源和最后核验日期；
  • 在价格、地区、限额等易变化信息上优先链接官方实时页面，而非把暂时数字包装成长期结论。

我们不做什么

  • 不提供规避地区、身份、付款或账户限制的方法；
  • 不收集读者的账户凭据、支付信息或身份证明；
  • 不把单一商业推荐包装成唯一“官方路径”。

需要账户、账单或资格确认时：请以 Claude 官方支持中心和结账页面为准；本站仅作独立信息整理。
