推广披露

读者应当一眼知道内容与商业关系之间的边界。

最后核验：2026-07-10

当前 GitHub Pages 站点不放置付费推广或佣金链接。如未来增加，本站会在链接附近和页面顶部清楚标注商业关系，不会以“官方”“唯一可用”或“保证成功”等措辞误导读者。

处理方式

  • 付费推广与佣金链接会以人类可读的文字披露；
  • HTML 链接会使用适当的 rel="sponsored" 标记；
  • 相关评测会同时写出限制、替代方案和不适用情形；
  • 不会推荐或销售用于规避地区、身份、支付或账户规则的服务。

若你认为某处披露不充分，请通过 GitHub 仓库 issue 提出，维护者会优先处理。
